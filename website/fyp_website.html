HAND GESTURE-CONTROLLED ROBOT CAR

OBJECTIVES

The goal of this project is to develop a robot car that can be remotely operated by gesture. In this project, the user can also operate the car's motions by equipping a controller hand glove and making specific gestures. Moreover, the robot car will ensure real-time response such as reduce time delay by improving the system, ensuring that the robot car will reacts quickly to the recognised hand signals. Reduce processing time and attaining real-time responsiveness by using effective algorithms and methodologies.

MATERIAL USED

18650 Li-ion Battery
ADXL335 Accelerometer
Piezo Buzzer
ESP32
L298N Motor Driver
Ultrasonic Sensor
TT Gear Motor
DC Wheel
Jumper Wires

STEPS

  1. Cutting the the base plate (acrylic board)
  2. Hot glue gunning the gear motor
  3. Attach the L298N motor driver to the downside of the board
  4. Solder some wire to connect the motor driver
  5. Installing the ESP32 and connecting to the motor driver
  6. Attach 18650 li-ion battery to power up
  7. Install ultrasonic sensor and buzzer
  8. Assemble transmitter device with accelerometer and ESP32
  9. Input algorithms to activate it
